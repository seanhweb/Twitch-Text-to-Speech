Introduction of Ads

Hi everyone, first off I want to say thank you for using the tool. A lot of you are, and it makes me happy that something I built is being used by many people. However, the cost of operation has become high, specifically around Amazon Polly.

As you can see in the graph above, the costs of running Amazon Polly are increasing every month. I'm left with three options:

  1. Increase Revenue
  2. Remove Amazon Polly
  3. Eat the cost

As it stands right now I can afford to keep running this, however if the costs become too great I can't. With that said, I'm preparing the website to display ads via Google Adsense. I can't ask you to turn off your adblocker, but I would greatly appreciate it. If you'd like to make a donation to the project, you can do so with the button below.

Again, thank you so much for using this tool!
