Removal of Amazon Polly and Tips

Hi all, my bills for Amazon Polly alone reached up to $80 / month. I hate to do this, but unforuntately I have to kill Amazon Polly until I can find a better method to synthesize speech. My costs for bandwidth and otherwise are relatively cheap, it's Amazon Polly that is the bulk of the costs.

With that said, for tax reasons and other personal reasons, I turned off tips. I appreciate your support, but the amounts I received were not enough to sustain the project.

I understand this is frustrating for many of you that used the Amazon Polly functionality. The good news is that the code is public on my github repository, so if you have your own Amazon Polly API key you are welcome to pay for it and host the project yourself.

Introduction of Ads

Hi everyone, first off I want to say thank you for using the tool. A lot of you are, and it makes me happy that something I built is being used by many people. However, the cost of operation has become high, specifically around Amazon Polly.

As you can see in the graph above, the costs of running Amazon Polly are increasing every month. I'm left with three options:

  1. Increase Revenue
  2. Remove Amazon Polly
  3. Eat the cost

As it stands right now I can afford to keep running this, however if the costs become too great I can't. With that said, I'm preparing the website to display ads via Google Adsense. I can't ask you to turn off your adblocker, but I would greatly appreciate it. If you'd like to make a donation to the project, you can do so with the button below.

Again, thank you so much for using this tool!
