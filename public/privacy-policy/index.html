Privacy Policy

Effective Date: 2/9/23

This Privacy Policy explains what data is collected and how it used when visiting the website or using this tool.

When Visiting any Page on the Website

We use Google Analytics to get a general idea of how the site is performing. For more information about how Google Analytics collects and uses data when you use our website, visit www.google.com/policies/privacy/partners, and to opt-out of Google Analytics, visit tools.google.com/dlpage/gaoptout.

When Making a Tip

When you send a monetary tip, your credit card information is processed via Stripe. Stripe saves the following information:

  • Full Name
  • Zip code
  • Last 4 digits of card used for payment
  • Email Address

When Messages are Spoken

When you input a twitch channel, the channel name does not leave your browser. In addition, messages spoken without Amazon Polly enabled are processed entirely within your browser. However, if the option to use Amazon Polly is checked, each message that is spoken is saved in mp3 format in an AWS S3 bucket. Each message is kept for a maximum of 24 hours, and is deleted from the bucket automatically.
